# AI Financial Analyst on GCP and Databricks

This repository contains an agentic financial analysis pipeline that retrieves SEC filings, extracts KPIs with Gemini on Vertex AI, validates the results, benchmarks against peer companies, generates investment briefs, logs runs to MLflow, and writes outputs to Databricks tables.
